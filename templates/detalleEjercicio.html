¿Qué significa el estado estacionario?

El estado estacionario muestra la distribución de probabilidad a la que tiende el sistema independientemente de su estado inicial, después de un gran número de transiciones.

Esta es la distribución de estados donde la máquina pasará la mayor parte de su tiempo a largo plazo, por lo que es clave para planificar el mantenimiento preventivo.

Recomendación a largo plazo:

{% set estacionario = resultado['estado_estacionario'] %} {% if estacionario[3] > 0.2 %}

{% elif estacionario[2] > 0.3 %}

A largo plazo, la máquina muestra tendencia a degradarse ({{ "%.1f"|format(estacionario[2] * 100) }}% en estado degradado). Se recomienda implementar un programa de mantenimiento preventivo recurrente.

{% elif (estacionario[0] + estacionario[1]) > 0.6 %}

A largo plazo, la máquina muestra un comportamiento estable con {{ "%.1f"|format((estacionario[0] + estacionario[1]) * 100) }}% de probabilidad de mantenerse en estados óptimo o normal. Se recomienda mantener el régimen actual de mantenimiento.

{% else %}
